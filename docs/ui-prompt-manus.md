# Manus UI 生成引导词 — 储能 EMS 运行看板（DeltaGrid EM）

> 将下文整段复制到 Manus，用于生成与当前代码实现一致的储能系统运行监控页面。默认生成**暗黑模式**主界面，并保留浅色模式切换能力说明。

---

## 一、产品定位

设计一套 **B 端储能虚拟电厂运行监控看板**（桌面 Web，≥1280px），面向运维值班人员。核心目标：一屏掌握储能运行状态、设备健康、充放电趋势与告警分布。风格参考现代工业 SaaS（Linear / Vercel 暗色仪表盘），信息密度高但层次清晰。

**页面名称**：储能系统运行看板  
**当前选中场景**：运行 → 储能系统  
**示例电厂**：钱塘江源网荷储虚拟电厂（浙江杭州）

---

## 二、页面框架（自上而下）

### 全局容器
- 最大宽度 1600px，居中，内边距 16–24px
- 页面背景：`#0b0f14`（暗色 canvas）/ 浅色 `#f0f2f5`
- 模块间距：16–20px 垂直栈
- 所有业务模块为**白底/深灰卡片**（圆角 8px、1px 边框、轻阴影）

### 布局栅格（4 行）

| 行 | 左列 (13fr) | 右列 (7fr) | 等高策略 |
|---|---|---|---|
| R1 | 储能信息 | 设备状态 | 左右等高拉伸 |
| R2 | 储能充放电曲线 | 电池一致性分析 | 右列固定高度 ~27rem，内容区内部滚动 |
| R3 | 储能充放电量 | 设备告警统计 | 左右等高 |

大屏 ≥1024px 为双列；小屏单列堆叠。

### 浮层（独立于栅格）
- **右上角固定**：SOC 紧急告警条（条件显示，可关闭）
- **顶栏右侧**：今日消息 hover 面板（450px 高）

## 三、模块功能与内容

### 3.1 顶栏（单卡片，内含主导航）

**第一行（左右分布）**

左侧：
- 电厂名称 H1 + 下拉切换（3 个虚拟电厂，含地区副标题）
- 胶囊标签「虚拟电厂」（蓝）
- 胶囊标签「正常运行中」（绿点 + 绿）
- 数据更新时间 + 刷新图标按钮（点击后旋转 600ms）

右侧（从左到右）：
- **消息铃铛**：红色角标显示未读数；hover 在按钮**下方**弹出面板
- **主题切换**：月亮/太阳图标 32×32

**第二行：主 Tab**（底部 2px 蓝线标识当前「运行」）  
场站总览 | 新能源收益 | **运行** | 台账 | 功率因数 | 储能测算 | 能效

**第三行：子 Tab**（圆角底纹，当前「储能系统」高亮蓝）  
光伏系统 | **储能系统** | 充电系统 | 单线图

---

### 3.2 今日消息面板（hover 触发）

- 宽度：22rem（max 视口宽 - 2rem）
- 高度：450px，列表区可滚动
- 位置：铃铛正下方，带**向上尖头**（与面板顶边无缝衔接，仅上/左描边）
- 标题「今日消息」+ 汇总「共 N 条未处理」
- 三类计数胶囊：告警(红) / 警告(橙) / 提示(蓝)
- 列表按级别分组，每条：标题、设备名、时间

---

### 3.3 SOC 紧急告警（条件浮层，右上）

触发：SOC ≤ 20% 且未关闭  
- 固定右上，宽 ≤360px，红框白/深灰底，自右滑入动画
- 标题「AI 紧急处置」+ P0 标签
- 文案含当前 SOC 数值、强制充电说明
- 按钮：「确认已知晓」「查看处置建议」
- 关闭 X

---

### 3.4 储能信息（左上一）

**左 38%：源网荷储拓扑图（嵌入，无独立外框）**
- SVG 拓扑：光伏(上) → 换电站/负荷(中) → 母线 → 储能 ESS(左下) + 电网(右下)
- 蓝色虚线动画能流：储能 → 母线 → 电网
- 节点 hover：**仅显示详情气泡**（无节点底色高亮框）
  - 光伏/负荷：气泡在节点**下方**，尖头朝上
  - 储能/电网：气泡在节点**上方**，尖头朝下；储能气泡右偏，不超出 cell
- 节点旁显示实时功率 kW

**右 62%：KPI 网格（2×2）**
1. **当前功率**：-1.30 kW（32px 粗体）+ 橙色「放电中」脉冲胶囊
2. **今日充/放电量**：12.50 / 2.10 kWh
3. **累计等效循环次数**：2
4. **SOC 电池 gauge** + **累计综合效率环图** 60.12%
   - 效率 <70% 时显示「AI 效能诊断」按钮（橙边浅底）
   - hover/展开：白底气泡 + 诊断报告 +「一键开启」绿按钮（气泡在按钮上方，带向下尖头）

---

### 3.5 设备状态（右上一）

- 标题「设备状态」
- **筛选胶囊 Tab**：全部 | 告警 | 需关注 | 正常 | AI预警（带数量，选中 halo 光晕）
- 设备卡片列表（示例展示 2 个设备组），每卡：
  - 设备名 + 状态统计徽章
  - 告警/需关注条目（标题+时间，默认第一条可见）
  - 「展开 (n)」合并展开：其余问题 + AI 预测提醒
  - AI 预测区块：蓝色左边框 cell

---

### 3.6 储能充放电曲线（左中）

- 标题 + 日期 `< 2025/08/21 当前 >` 导航
- 组合图：面积+折线，X=时间，左 Y=功率 kW，右 Y=SOC %
- 充电蓝 `#3b82f6`、放电橙 `#f59e0b`；功率过零插值
- 背景 ReferenceArea 标示电价时段（谷/峰/尖/深谷）
- 图例底部：充放电功率、SOC、电价时段色块
- hover tooltip；**点击不出现持久高亮边框**

---

### 3.7 电池一致性分析（右中，固定高度模块）

- 标题 + 筛选 Tab：全部 | 告警 | 需关注 | AI预警
- **快捷 Chip**（横向）：每类最多 1 个代表簇，选中 halo
- **搜索下拉**：选电池簇 RACK-C1 · 全名
- 选中簇详情：
  - 指标行：电压状态 | 最高/最低电压 | 压差 | 温度（竖线分隔）
  - 告警/需关注：**问题分析&解决建议** 正文
  - AI 预测：蓝框 cell，「展开 AI 劣化趋势」→ 内嵌 mV 折线图 + 临界 15mV 红线
- 列表区超出模块高度时**内部滚动**

---

### 3.8 储能充放电量（左下）

- 标题 + 日/月/年 分段 + 日期导航
- 双柱图：蓝=充电量、橙=放电量，Y=kWh
- hover 10% 白行背景；点击不残留边框
- 底部图例

---

### 3.9 设备告警统计（右下）

- 标题 + 提示「点击柱状条可查看告警详情」
- 勾选「隐藏无告警日期」
- 日/月/年 + 周期导航
- **横向堆叠条形图**（Y=日期，X=次数），设备分色：
  - 逆变器红、烟感深红、采集器橙、环测仪黄、温湿度橙红、摄像头蓝
- hover 行 10% 白底 + tooltip
- **点击柱条**：右侧滑出告警详情面板（min-h 200px，宽 288–320px），局部蒙层；点击面板外关闭
- 底部图例为**开关**（可隐藏某类设备柱）

## 四、交互逻辑清单

| 交互 | 行为 |
|---|---|
| 虚拟电厂下拉 | 点击展开列表，选后更新标题，点遮罩关闭 |

| 拓扑节点 hover | 仅气泡，无底色框 |

| 设备状态 Tab | 筛选设备列表 |
| 设备展开 | 展开显示全部问题+AI预测 |
| 电池 Tab/Chip/下拉 | 联动筛选与选中簇 |

## 五、视觉要素（Design Tokens）

### 色彩
```
Canvas 暗: #0b0f14    Canvas 亮: #f0f2f5
Card  暗: #151b26    Card  亮: #ffffff
Success: #22c55e    Warning: #f59e0b
Danger:  #ff3b30    Info:    #3b82f6
Charge:  #3b82f6    Discharge: #f59e0b
Muted:   #64748b / #8b9cb3(暗)
```

### 字体
- 西文 Inter + 中文 PingFang SC / 微软雅黑
- KPI 一级 32px/700；二级 18px/600；三级 12px/400
- 卡片标题 14px/600；图表轴 10px 斜体 `#999` / `#6b7c93`

### 组件风格
- 卡片：圆角 8px，border slate-200 / slate-700
- 胶囊 Tab：圆角 full，浅底+色边，选中加深+halo
- 分段器：浅灰底 + 白底选中块
- 图表：虚线网格、无轴线、hover 轻高亮
- 气泡：白/深灰底、8px 圆角、轻阴影、三角尖头与边框同色

### 图标
- Lucide 风格线性图标：Bell, Moon, Sun, RefreshCw, ChevronDown, Bot 等
